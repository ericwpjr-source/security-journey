### TACACS+

**TACACS+** stands for **Terminal Access Controller Access-Control System Plus**.

It is an authentication protocol often used as an alternative to RADIUS for managing administrative access to network devices.

#### Security Benefits

- TACACS+ encrypts the authentication packet body.
- RADIUS typically encrypts only the user's password.
- TACACS+ separates authentication, authorization, and accounting.
- This allows more detailed control over administrator commands and permissions.

#### Key Takeaway

TACACS+ is commonly used for administrative access to routers, switches, and other network devices because it provides detailed authorization and stronger protection of the authentication exchange.
